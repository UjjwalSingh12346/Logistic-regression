#Explanation of the Streamlit App Code:
Model Loading:

model1 and model2 are loaded using joblib.load() to make the trained models available in the app.
Input Fields:

We use st.number_input() and st.selectbox() to create input fields where users can enter the data for prediction.
The app collects values for features like Pclass, Age, Sex, etc., for both datasets.
Prediction Functions:

predict_model1() and predict_model2() take the input features, pass them to the trained models (model1 and model2), and return the prediction.
